# 个人天赋挖掘与成长软件 (Python)

这是一个基于 Python + Streamlit 的个人成长工具，用于：

- 天赋测评（8 个维度）
- 自动生成 12 周成长计划
- 成长目标管理（可勾选完成）
- 每日状态打卡与趋势追踪
- 登录与多用户数据隔离
- 教练建议与连续互动对话
- 出生信息综合测算（星座 + 节气年柱/月柱 + 紫微主轴映射）

## 1. 安装依赖

在项目目录执行：

```bash
pip install -r requirements.txt
```

## 2. 启动应用

```bash
streamlit run app.py
```

默认会在浏览器打开本地页面。

## 3. 数据存储

- 程序会自动生成本地数据库：`talent_growth.db`
- 主要表：`users`、`assessments`、`goals`、`checkins`、`coach_messages`
- 新增：`birth_profiles`（保存出生信息并用于命理建议）
- 所有核心业务数据按 `user_id` 隔离

## 5. 命理测算说明

- 在“天赋测评”页可输入出生日期、出生时分、出生地经纬度
- 系统自动识别 IANA 时区（含历史夏令时偏移），无需手填 UTC 偏移
- 系统先做真太阳时修正（经度修正 + 均时差）
- 年柱/月柱按节气边界计算（含立春切换）
- 系统会给出：星座、生肖、年柱/月柱、五行倾向、适合方位、适合行业、十年大运节奏
- 该模块用于成长参考，不替代专业命理排盘

## 4. 后续可扩展方向
